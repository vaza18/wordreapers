# Відкритий код

**Wordreapers** (Словозбирачі) — проєкт з відкритим вихідним кодом.

## Навіщо

Гра народилась як сімейна традиція. Відкритий код дозволяє:

- переконатися, що додаток робить лише те, що заявлено (особливо щодо даних);
- запропонувати виправлення та покращення;
- адаптувати гру під свої потреби;
- зберегти гру, навіть якщо автор припинить активну підтримку.

## Ліцензія

Вихідний код додатку поширюється на умовах **[MIT License](../../../LICENSE)** (або іншої ліцензії, зазначеної на GitHub).

Коротко: можна використовувати, змінювати і поширювати код, зокрема у комерційних цілях, за умови збереження тексту ліцензії та відмови від гарантій.

> **Примітка:** ліцензія на _код_ не означає, що всі _дані_ в додатку вільні для копіювання. Словник [ВЕСУМ](https://github.com/brown-uk/dict_uk) має власну ліцензію CC BY — див. розділ «Сторонні матеріали» нижче.
